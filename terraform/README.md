# Terraform

This directory will contain Infrastructure as Code experiments and production-style patterns for NovaLab.

## Planned Targets

- Proxmox
- VMware
- Future cloud providers

## Learning Goals

- Terraform provider configuration
- Variables
- Outputs
- State management
- Terraform plan/apply workflow
- CI/CD validation using GitHub Actions
- Safe infrastructure automation practices

## Initial Focus

The first Terraform buildout will target Proxmox using a controlled lab workflow.

VMware support will be added later to compare infrastructure automation patterns between hypervisors.
